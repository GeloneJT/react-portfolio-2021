CoinFinder

CoinFinder is a Web App that takes an address or location data and runs them through google's geolocation api, gets the coordinates of the user and then runs that - with a search radius to CoinMap api and then obtains the locations within that radius, runs those locations through Google's Places Api and displays the locations on a google map. Using the CoinMap data to list the nearest 10 locations to the user along with many more locations on the map

Project Details

  • Uses location to find venues that transact with cryptocurreny
  • Google Maps and CoinMap APIs used
  • HTML/CSS/JavaScript/Zurb Foundation used
  • Finished phase 1 of developement
